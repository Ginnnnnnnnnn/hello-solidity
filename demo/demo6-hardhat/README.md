# hardhat 指令

- // 初始化
- npx hardhat --init
- // 运行
- npx hardhat run scripts/deployFundMe.js --network sepolia
- // 验证
- npx hardhat verify --network sepolia 0x8b354301e048D6c48A4b63281D9F74807b8a3C39 "参数"
- // 部署
- npx hardhat deploy --network sepolia --reset
